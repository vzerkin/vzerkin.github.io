## JSON Viewer/Editor

### Purpose/Features:
* Presenting **any** JSON text as interactive tree in order to learn/understand/compare/discuss data structures/content/hierarchy
* Test samples: nuclear data JSON files from EXFOR, ENDF, ENSDF, NSR, IBANDL, etc.
* Viewer-part is extended to display specific information along with node title
* Editor implements operations: 
  * File: open/new/save
  * JSON-text: modify/clear/copy/paste
  * Tree-node: move:up/down/remove
  * Edit: undo/redo
  * History: view history of operations roll back to previous steps
  * Tool: minify/expand/iTable current JSON text in pupup-window
* Application can be used in local Web-Browser - without Web-server
